# Preferences


## 基本的なマッピング

### ノーマルモード

###### 1ストローク
| map     | default                    | preference                                     |
|:-------:|:---------------------------|:-----------------------------------------------|
| `a`     | カーソルの後ろに挿入       |                                                |
| `A`     | 行末に挿入                 |                                                |
| `<C-a>` | インクリメント             | 次のハンクに移動                               |
| `b`     | word のはじめに後方移動    |                                                |
| `B`     | WORD のはじめに後方移動    |                                                |
| `<C-b>` | 1ページ上にスクロール      | git diff                                       |
| `c`     | change                     |                                                |
| `C`     | c\$                        |                                                |
| `<C-c>` | 特になし                   | バッファを閉じる                               |
| `d`     | delete                     |                                                |
| `D`     | d\$                        |                                                |
| `<C-d>` | 半ページ下にスクロール     | 半ページ上にスクロール                         |
| `e`     | word の終わりに前方移動    |                                                |
| `E`     | WORD の終わりに前方移動    |                                                |
| `<C-e>` | ページを下に一行スクロール |                                                |
| `f`     | 指定した char へ前方移動   | clever-f                                       |
| `F`     | 指定した char へ後方移動   | clever-F                                       |
| `<C-f>` | 1ページ下にスクロール      | 半ページ下にスクロール                         |
| `g`     | global prefix              |                                                |
| `G`     | 指定した行へジャンプ       | カーソル下の文字を右の文字と入れ替え           |
| `<C-g>` | ステータスを表示           | git status                                     |
| `h`     | カーソルを左へ移動         |                                                |
| `H`     | 画面の一番上の行に移動     | 行頭に移動（非空白）                           |
| `<C-h>` | カーソルを左に移動         | 画面の一番上の行に移動                         |
| `i`     | カーソルの前に挿入         |                                                |
| `I`     | 行頭に挿入（非空白）       |                                                |
| `<C-i>` | `<Tab>` と同じキーコード   | `<Tab>` と同じキーコード                       |
| `j`     | カーソルを下に移動         |                                                |
| `J`     | カーソル行と下の行を結合   | denite バッファの次の候補を実行                |
| `<C-j>` | 1行下に移動                | 次のバッファに移動                             |
| `k`     | カーソルを上に移動         |                                                |
| `K`     | カーソル下の word を man   | denite バッファの前の候補を実行                |
| `<C-k>` | ダイグラフの入力           | 前のバッファに移動                             |
| `l`     | カーソルを右へ移動         |                                                |
| `L`     | 画面の一番下の行に移動     | 行末に移動                                     |
| `<C-l>` | 画面を再描画               | 画面の一番下の行に移動                         |
| `m`     | マーク                     | word の終わりに後方移動                        |
| `M`     | 画面の中央に移動           | WORD の終わりに後方移動                        |
| `<C-m>` | `<CR>` と同じキーコード    | `<CR>` と同じキーコード                        |
| `n`     | 前方の検索結果に移動       | 前方の検索結果に移動して画面中央までスクロール |
| `N`     | 後方の検索結果に移動       | 後方の検索結果に移動して画面中央までスクロール |
