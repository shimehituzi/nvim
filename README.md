# Preferences


## 基本的なマッピング

### ノーマルモード

###### 1ストローク
| map                     | default                      | preference                                     |
|:-----------------------:|:-----------------------------|:-----------------------------------------------|
| `a`                     | カーソルの後ろに挿入         |                                                |
| `A`                     | 行末に挿入                   |                                                |
| `<C-a>`                 | インクリメント               | カーソル行が上から10行目になるようにスクロール |
| `b`                     | word のはじめに後方移動      |                                                |
| `B`                     | WORD のはじめに後方移動      |                                                |
| `<C-b>`                 | 1ページ上にスクロール        |                                                |
| `c`                     | change                       |                                                |
| `C`                     | 行末まで変更                 |                                                |
| `<C-c>`                 | 特になし                     | バッファを閉じる                               |
| `d`                     | delete                       |                                                |
| `D`                     | 行末まで削除                 |                                                |
| `<C-d>`                 | 半ページ下にスクロール       | 半ページ上にスクロール                         |
| `e`                     | word の終わりに前方移動      |                                                |
| `E`                     | WORD の終わりに前方移動      |                                                |
| `<C-e>`                 | ページを下に1行スクロール    |                                                |
| `f`                     | 指定した char へ前方移動     | clever-f                                       |
| `F`                     | 指定した char へ後方移動     | clever-F                                       |
| `<C-f>`                 | 1ページ下にスクロール        | 半ページ下にスクロール                         |
| `g`                     | global prefix                |                                                |
| `G`                     | 指定した行へジャンプ         | 大文字，小文字を切り替え                       |
| `<C-g>`                 | ステータスを表示             |                                                |
| `h`                     | カーソルを左へ移動           |                                                |
| `H`                     | 画面の1番上の行に移動        | 行頭に移動（非空白）                           |
| `<C-h>`                 | カーソルを左に移動           | 画面の1番上の行にカーソルを移動                |
| `i`                     | カーソルの前に挿入           |                                                |
| `I`                     | 行頭に挿入（非空白）         |                                                |
| `<C-i>`                 | `<Tab>` と同じキーコード     | `<Tab>` と同じキーコード                       |
| `j`                     | カーソルを下に移動           |                                                |
| `J`                     | カーソル行と下の行を結合     | denite バッファの次の候補を実行                |
| `<C-j>`                 | 1行下に移動                  | 次のバッファに移動                             |
| `k`                     | カーソルを上に移動           |                                                |
| `K`                     | カーソル下の word を man     | denite バッファの前の候補を実行                |
| `<C-k>`                 | ダイグラフの入力             | 前のバッファに移動                             |
| `l`                     | カーソルを右へ移動           |                                                |
| `L`                     | 画面の1番下の行に移動        | 行末に移動                                     |
| `<C-l>`                 | 画面を再描画                 | 画面の1番下の行にカーソルを移動                |
| `m`                     | マーク                       | word の終わりに後方移動                        |
| `M`                     | 画面の中央に移動             | WORD の終わりに後方移動                        |
| `<C-m>`                 | `<CR>` と同じキーコード      | `<CR>` と同じキーコード                        |
| `n`                     | 前方の検索結果に移動         | 前方の検索結果に移動して画面中央までスクロール |
| `N`                     | 後方の検索結果に移動         | 後方の検索結果に移動して画面中央までスクロール |
| `<C-n>`                 | 次の行に移動                 | ハンクをステージ                               |
| `o`                     | カーソル行の下に行を挿入     |                                                |
| `O`                     | カーソル行の上に行を挿入     |                                                |
| `<C-o>`                 | 前のジャンプ位置にジャンプ   |                                                |
| `p`                     | 前方にペースト               |                                                |
| `P`                     | 後方にペースト               |                                                |
| `<C-p>`                 | 前の行に移動                 | ペーストし，カーソルをペースト範囲の最後に移動 |
| `q`                     | マクロ登録                   |                                                |
| `Q`                     | Ex モードに切り替え          | マクロの再生                                   |
| `<C-q>`                 | ビジュアル矩形モード         | tmux の prefix                                 |
| `r`                     | 1文字置き換え                |                                                |
| `R`                     | 置き換えモード               |                                                |
| `<C-r>`                 | やり直し                     |                                                |
| `s`                     | 1文字削除して挿入モード      | default + レジスタに保存しない                 |
| `S`                     | 1行削除して挿入モード        | WORD に対し Surround                           |
| `<C-s>`                 | 特になし?                    | git status                                     |
| `t`                     | 指定した char の前へ前方移動 | インクリメント                                 |
| `T`                     | 指定した char の後へ後方移動 | ディクリメント                                 |
| `<C-t>`                 | 次のタグにジャンプ           |                                                |
| `u`                     | 取り消し                     |                                                |
| `U`                     | 変更ありの取り消し           | カーソル行と下の行を結合                       |
| `<C-u>`                 | 半ページ上にスクロール       | ハンク全体をUndoする．                         |
| `v`                     | ビジュアルモード             |                                                |
| `V`                     | 1行ビジュアルモード          |                                                |
| `<C-v>`                 | 矩形ビジュアルモード         |                                                |
| `w`                     | word のはじめに前方移動      |                                                |
| `W`                     | WORD のはじめに前方移動      |                                                |
| `<C-w>`                 | ウィンドウ関連の prefix      |                                                |
| `x`                     | カーソル下の文字を削除       |                                                |
| `X`                     | カーソルの前の文字を削除     |                                                |
| `<C-x>`                 | ディクリメント               | git diff                                       |
| `y`                     | yank                         |                                                |
| `Y`                     | 1行ヤンク                    | 行末までヤンク                                 |
| `<C-y>`                 | ページを上へ1行スクロール    |                                                |
| `z`                     | prefix                       |                                                |
| `Z`                     | prefix                       |                                                |
| `<C-z>`                 | vim をサスペンド             |                                                |
| `<Esc>`                 | `<Esc>`                      | `<Esc>`                                        |
| `<Tab>`                 | 次のジャンプ位置にジャンプ   | 指定した行へジャンプ                           |
| `<S-Tab>`               | 特になし?                    | 次のジャンプ位置へジャンプ                     |
| `<BS>`                  | カーソルを左に移動           | 前のジャンプ位置へジャンプ                     |
| `<CR>`                  | 下の行頭の非空白文字に移動   | 保存                                           |
| `<Space>`               | カーソルを右に移動           | 個人的なマッピングの prefix                    |
| `!`                     | 外部のプログラムを実行       |                                                |
| `@`                     | マクロの再生                 | ファイルタイプ別のマッピングをするため `<nop>` |
| `#`                     | カーソル下の単語を後方検索   | ファイルタイプ別のマッピングをするため `<nop>` |
| `$`                     | 行末に移動                   | ファイルタイプ別のマッピングをするため `<nop>` |
| `%`                     | 対応するかっこに移動         |                                                |
| `^`                     | 行頭に移動（非空白）         | カーソル下の単語を含む単語をハイライト         |
| `&`                     | 置き換えを繰り返す           | `^`, `*` のハイライト箇所を置き換え            |
| `*`                     | カーソル下の単語を前方検索   | カーソル下の単語をハイライト                   |
| `(`                     | 文後方に移動                 |                                                |
| `)`                     | 文前方に移動                 |                                                |
| `-`                     | 上の行頭の非空白文字に移動   | 次のハンクに移動                               |
| `_`                     | -1 行下の非空白文字に移動    | 前のハンクに移動                               |
| `=`                     | インデントを揃える           |                                                |
| `+`                     | 下の行頭の非空白文字に移動   | git gutter をトグル                            |
| `\`                     | デフォルト `<LocalLeader>`   | （よくプラグインで勝手にマッピングされる）     |
| <code>&#124;</code>     | カーソル行の指定した列へ移動 | ネストを可視化するプラグインをトグル           |
| <code>&#096;</code>     | マーク位置へジャンプ         | ハイライトを消す                               |
| `~`                     | 大文字，小文字を切り替え     | カーソル下の文字と右の文字を入れ替える         |
| `[`                     | 後方移動系コマンドの prefix  |                                                |
| `]`                     | 前方移動系コマンドの prefix  |                                                |
| `{`                     | 段落後方に移動               |                                                |
| `}`                     | 段落前方に移動               |                                                |
| `;`                     | `f cahr` の順方向繰り返し    | 画面上の任意の行にジャンプ                     |
| `:`                     | コマンドラインモード         |                                                |
| `'`                     | マーク位置の行頭へジャンプ   | マーク位置へジャンプ                           |
| `"`                     | レジスタの指定               |                                                |
| `,`                     | `f char` の逆方向繰り返し    | 画面上の任意の位置にジャンプ                   |
| `.`                     | 繰り返し                     |                                                |
| `<`                     | インデントを左にずらす       |                                                |
| `>`                     | インデントを右にずらす       |                                                |
| `/`                     | 前方検索                     |                                                |
| `?`                     | 後方検索                     |                                                |
| `<Down>`                | カーソルを下に移動           | 下のウィンドウに移動                           |
| `<Up>`                  | カーソルを上に移動           | 上のウィンドウに移動                           |
| `<Right>`               | カーソルを右に移動           | 右のウィンドウに移動                           |
| `<Left>`                | カーソルを左に移動           | 左のウィンドウに移動                           |
| `<S-Down>`              | 1ページ下にスクロール        | ウィンドウを下に移動                           |
| `<S-Up>`                | 1ページ上にスクロール        | ウィンドウを上に移動                           |
| `<S-Right>`             | word のはじめに前方移動      | ウィンドウを右に移動                           |
| `<S-Left>`              | word のはじめに後方移動      | ウィンドウを左に移動                           |
| `<C-Down>`              | 特になし                     | Mac の制御に使っているため無効                 |
| `<C-Up>`                | 特になし                     | Mac の制御に使っているため無効                 |
| `<C-Right>`             | WORD のはじめに前方移動      | Mac の制御に使っているため無効                 |
| `<C-Left>`              | WORD のはじめに後方移動      | Mac の制御に使っているため無効                 |
| `<C-S-Down>`            | 特になし                     | 現在のウィンドウの高さを狭く                   |
| `<C-S-Up>`              | 特になし                     | 現在のウィンドウの高さを広く                   |
| `<C-S-Right>`           | 特になし                     | 現在のウィンドウの幅を狭く                     |
| `<C-S-Left>`            | 特になし                     | 現在のウィンドウの幅を広く                     |
| `<C-@>`                 | `<C-Space>` と同じキーコード | ページを上へ1行スクロール                      |
| `<C-#>` `<C-[>`         | `<esc>` と同じキーコード     | `<esc>` と同じキーコード                       |
| `<C-$>` `<C-\>`         | 特になし?                    | 1個前のファイルに戻る                          |
| `<C-%>` `<C-]>`         | カーソル下の定義にジャンプ   |                                                |
| `<C-^>`                 | 1個前のファイルに戻る        |                                                |
| `<C-&>` `<C-->` `<C-/>` | 特になし?                    |                                                |
| `<C-*>`                 | `<BS>` と同じキーコード      | `<BS>` と同じキーコード                        |
