### ノーマルモード

###### 1ストローク
| map     | default                    | preference                           |
|:-------:|:---------------------------|:-------------------------------------|
| `a`     | カーソルの後ろに挿入       |                                      |
| `A`     | 行末に挿入                 |                                      |
| `<C-a>` | インクリメント             | 次のハンクに移動                     |
| `b`     | word のはじめに後方移動    |                                      |
| `B`     | WORD のはじめに後方移動    |                                      |
| `<C-b>` | 1ページ上にスクロール      | git diff                             |
| `c`     | change                     |                                      |
| `C`     | c\$                        |                                      |
| `<C-c>` | 特になし                   | バッファを閉じる                     |
| `d`     | delete                     |                                      |
| `D`     | d\$                        |                                      |
| `<C-d>` | 半ページ下にスクロール     | 半ページ上にスクロール               |
| `e`     | word の終わりに前方移動    |                                      |
| `E`     | WORD の終わりに前方移動    |                                      |
| `<C-e>` | ページを下に一行スクロール |                                      |
| `f`     | 指定した char へ前方移動   | clever-f                             |
| `F`     | 指定した char へ後方移動   | clever-F                             |
| `<C-f>` | 1ページ下にスクロール      | 半ページ下にスクロール               |
| `g`     | global prefix              |                                      |
| `G`     | 指定した行へジャンプ       | カーソル下の文字を右の文字と入れ替え |

| `k`     | カーソルを上に移動         |                                      |
| `K`     | カーソル下の word を man   | denite バッファの前の候補を実行      |
